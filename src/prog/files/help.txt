БИЛЛИАРДНЫЙ КЛУБ
------------------------------------------
• Управление заказами биллиардных столов в клубе.
• В клубе биллиардные столы двух типов, они отличаются диаметром луз: русский биллиард и снукер.
• При запуске программы заказы загружаются из файла.
  При выключении программы(соотвествующей командой меню) заказы сохраняются в файл
  Для каждого стола используется свой файл заказов.

ТЕХНОЛОГИИ:
• LocalTime: ввод и вывод на экран с использованием шаблонов, рассчет длительности временного интервала,
  проверка вхождения времени во временной интервал, компаратор по возрастанию времени
• Файлы: сохранение в файл и чтение из файла с помощью сериализации, построчное чтение текстового файла
